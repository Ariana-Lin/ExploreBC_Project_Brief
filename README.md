# Explore BC – Responsive Tourism Website

## Project Overview
This is a starter code package for the **Explore BC** project in ** ACIT 1620 – Fundamental Web Technologies**.

You will build a multi-page responsive website promoting travel destinations in British Columbia using HTML5 and CSS3.

---

## File Structure
```
ExploreBC_Project/
├── index.html
├── destinations.html
├── contact.html
├── css/
│   └── style.css
└── images/
```
- **index.html** – Home page with hero banner and navigation.
- **destinations.html** – Gallery of travel destinations using Grid or Flexbox.
- **contact.html** – Contact form with styled inputs.
- **css/style.css** – External stylesheet for all pages.
- **images/** – Placeholders for your project images.

---

## Setup Instructions
1. Extract the ZIP file into a local folder.
2. Open the project in **Visual Studio Code** or your preferred editor.
3. Open `index.html` in a browser to preview the site.
4. Edit each HTML file to add your own content and images.
5. Update `style.css` to design and style your website.
6. Test responsiveness by resizing the browser window or using DevTools.
